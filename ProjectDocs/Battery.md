## 电池状态数据

主题：CSDK/Battery/state/listener

参数：

| 参数名称 | 参数类型 | 参数描述 |
| :------: | :------: | :------: |
| batteries | [] | 电池状态数组 |
| index | int | 电池索引 |
| capacity_percent | int | 电量百分比 |
| temperature | double | 电池温度 |
| voltage | float | 电压/mV |
| number_of_discharges | int | 循环次数 |
| is_being_charged | boolean | 是否在充电 |

描述：电池状态数据监听
